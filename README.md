Features
*Get random companies from API
*Filter by date
*Search using autocomplete
*Notify user when offline

Technology Stack
*State Management - Bloc
*Flutter-type ahead for handling autocomplete
*Market stack REST API
*Connectivity plus for handling offline and online presence
*Dart , Flutter
